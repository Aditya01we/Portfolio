🚀 Modern Developer Portfolio

A fully responsive and interactive Developer Portfolio Website built with React.js, JavaScript, CSS3, and Vite. This portfolio showcases my professional journey, technical skills, projects, certifications, education, experience, and contact information through a modern, user-friendly interface with smooth animations and optimized performance.

Designed with a clean UI/UX approach, the website provides an engaging experience across desktop, tablet, and mobile devices while highlighting my frontend development expertise.

📂 Installation

Clone the repository:

git clone https://github.com/your-username/Portfolio.git

Navigate to the project directory:

cd Portfolio

Install dependencies:

npm install

Run the development server:

npm run dev

Create a production build:

npm run build

Preview the production build:

npm run preview
